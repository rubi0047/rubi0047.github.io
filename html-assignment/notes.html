5 HTML tags

<p>
The <p> tag in HTML stands for paragraph and is used to define blocks of text that are separated by margins. It is one of the most common and basic tags for formatting text on a webpage.
<ol>
The <ol> tag in HTML stands for ordered list and is used to create a list of items where the order is important. Each item in the list is marked with a number or another form of sequencing like letters or Roman numerals.
<li>
The <li> tag in HTML stands for list item and is used to define individual items within a list. It can be used inside both ordered lists and unordered lists, and it represents one item within that list.
<ul>
The <ul> tag in HTML stands for unordered list and is used to create a list of items where the order of the items is not important. Items in an unordered list are typically marked with bullets or other symbols, rather than numbers or letters.
<img>
The <img> tag in HTML is used to embed images in a web page. It is an empty tag, meaning it doesn't have a closing tag and is self-contained. The <img> tag requires at least one attribute to define the image source.
